# GradlePluginExample

## Installation

You can use every Gradle project to test this Plugin.


#### Kotlin DSL

```
plugins {
    id("my.project.group.plugin") version "1.0.0"
}
```

#### Groovy DSL

```
plugins {
    id 'my.project.group.plugin' version '1.0.0'
}
```

## Usage

```
./gradlew my-task
```
